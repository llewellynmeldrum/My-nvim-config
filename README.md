# My nvim config
Modularized fork of kickstart nvim, with a few custom keybinds. 


## Custom keybinds

| Name                 | Keybind            | 
| -------------------- | -------------------|
| Accept autocomplete  |  `<C-CR>`/`<S-CR>` |
| Goto manual <br><br> |  `gm`              |
| Change theme<br>     |  `ct`              |
| Toggle zen mode      |  `<leader>zm`      |
| Toggle reading mode  |  `<leader>rm`      |
