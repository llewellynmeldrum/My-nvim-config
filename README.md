# My nvim config
Modularized fork of kickstart nvim, with a few custom keybinds. 


## Custom keybinds

| Name                 | Keybind                         | NVIM style        |
| -------------------- | ------------------------------- | ----------------- |
| Accept autocomplete  | `CTRL+RETURN` OR `SHIFT+RETURN` | `<C-CR>`/`<S-CR>` |
| Goto manual <br><br> | `gm`                            | `gm`              |
| Change theme<br>     | `ct`                            | `ct`              |
| Toggle zen mode      | `[SP]zm`                        | `<leader>zm`      |
| Toggle reading mode  | `[SP]rm`                        | `<leader>rm`      |
